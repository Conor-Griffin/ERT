Fire classes:

  • Class A: solid combustibles (wood, paper, fabric)
  • Class B: flammable liquids (gasoline, oil, paint)
  • Class C: flammable gases (propane, butane, methane)
  • Class D: combustible metals (magnesium, titanium, aluminum)
  • Class E: Electrical fires involving live wires
  • Class F: cooking oils and fats

Fire extinguishers:

  • Follow the PASS technique: Pull, Aim, Squeeze, Sweep
  • Water: for Class A fires only
  • Foam: for Class A and B fires
  • Dry Powder: for Class A, B, C, D and E fires
  • CO2: for Class B and E fires
  • Wet Chemical: for Class F fires

Firefighting procedure (RECEOVES):

  1. Rescue
  2. Exposures
  3. Contain
  4. Extinguish
  5. Overhaul
  6. Ventilate
  7. Environment
  8. Salvage

When is BA required? (HOTS):

  • Heated atmosphere
  • Oxygen deficient
  • Toxic atmosphere
  • Smoke/suspended particles

Rules of BA (METOPDOG):

  • M - Minimum 250bar
  • E - Report to ECO before entry
  • T - Teams of 2
  • O - OIC must authorise
  • P - PDA's take priority
  • D - Doff and don in fresh air
  • O - One out, all out
  • G - Guidelines to be considered

Working BA duration = cylinder pressure (bar)- 50bar (safety margin) x cylinder volume / 60L/min.
(300-50) x 6.8 / 60 = 28mins
